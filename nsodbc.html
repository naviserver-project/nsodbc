Theory of Operation

This is an internal database driver that connects to ODBC databases on Win32 only.  Because it requires external ODBC client libraries, Unix users will need to obtain those drivers from the database vendor.

Known Issues

Works on Win32 only.

Unix users can use UnixODBC from http://genix.net/unixODBC/or FreeODBC from http://www.freeodbc.org/.  Send us your changes so we can include them in the AOLserver distribution!
